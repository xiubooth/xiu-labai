# Bundled Claw Runtime Asset

- Platform: Windows x64
- Binary: `claw.exe`
- Intended managed install location: `%LOCALAPPDATA%\LabAI\runtime\claw\claw.exe`
- Purpose: ordinary RA release installs should not require Rust, cargo, or a local `claw-code` source checkout

This asset is bundled for the Phase 17 Windows release bootstrap. Developer source builds remain available separately through the existing advanced scripts and docs, but they are not the default release path.
